TO RUN:

NPM install at root (/spotify_firebase_integration/)

add files app.js and serviceAccountKey.json in the subdir authorization_code
(note that both these files have secret info so be aware of that)

to run: in terminal, in the subdir authorization_code, type:
node app.js

^if this works, it should log 'listening at port 8888'
go to localhost:8888 in browser to see the page
